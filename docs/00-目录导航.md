# Hermes-mini 教程导航

> 每个文件对应一章，代码与文档一一对应。

## 学习路线

```
第 0 章（必读）  →  你在这里
第 1 章          →  Agent Loop：感知 → 推理 → 行动
第 2 章          →  记忆系统：工作记忆 + 长期记忆 + 用户画像
第 3 章          →  工具系统：注册 → 调用 → 结果注入
第 4 章          →  自进化 Skill：任务后自动生成，下次直接调用
第 5 章          →  多平台 Gateway：CLI + Telegram，统一会话管理
第 6 章          →  完整 Agent：整合所有模块，一条命令跑起来
```

## 每章结构

每个章节包含：
- 📖 **原理解析**（`docs/XX-xxx.md`）—— 讲清楚"为什么这样设计"，不只说"是什么"
- 💻 **配套代码**（`src/`）—— 可独立运行，边读边改
- 🧪 **练习题**（`tests/`）—— 学完本章后自己动手验证理解

## 如何学习

1. 先通读对应 `.md` 文件
2. 跑通对应的 `src/` 代码
3. 做练习题
4. 进入下一章

> 不要跳章！每一章的代码都依赖前一章的结论。

## 常见问题

**Q: 需要 GPU 吗？**
不需要。所有模型调用走 API，本地只跑 Python。

**Q: 支持哪些模型？**
MiniMax（默认）、GLM-5、Qwen、OpenAI。换模型只需改一行配置。

**Q: 代码风格是什么？**
中文注释为主，变量/函数名用英文（业界惯例），代码简洁不过度封装。
